github pages

首先使用github pages ，创建基于项目，用来存放静态的网页、博文 github pages 提供模板功能，可以创建和编辑静态的网页来生成文章;如何自动创建文章

安装jekyll

jekyII是静态网页发布引擎，编辑和上传网页源码后，github会通过jekyII对源码进行处理并生成静态文件； 使用jekyII可以简单快速的发布文章，而不用每次都麻烦的按照html格式编辑并生成网页

安装前的requirement检查

  1. ruby 建议 1.9.3版本及以上
  2. RubyGems
  3. Linux, Unix, or Mac OS X

安装||更新 ruby

  1. ruby -v 检查版本，如果版本低于1.9.3，需要安装新版本的ruby
  2. 下载安装rvm，来管理多版本的ruby curl -L https://get.rvm.io | bash -s stable --ruby
  3. rvm list known 查看所有可用的最新版本的ruby
  4. rvm install ruby-2.1.0 安装2.1.0版本的ruby；安装过程中如果出现“Can not find compiler and 'make' tool - make sure Xcode and/or Command Line Tools are installed.”，参考http://stackoverflow.com/questions/17006304/why-cant-rvm-find-compiler-and-make-tool-after-installing-xcode，安装command Line Tools
  5. rvm use ruby-2.1.0 切换使用2.1.0版本的ruby

安装 jekyll

gem install jekyll

使用jekyll 开始新博客之旅

  1. jekyll new blogname
  2. cd blogname
  3. jekyll serve
  4. 本地访问 http://localhost:4000
